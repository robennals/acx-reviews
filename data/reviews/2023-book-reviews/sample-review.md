# Sample Review

This is a sample review to test the ACX Reviews app. The actual content will come from the Google Docs and ACX Substack posts.

## How to populate with real content

1. Run `npm run fetch-acx` to scrape ACX Substack posts
2. Run `npm run fetch-gdocs` to extract from Google Docs (requires API setup)
3. Run `npm run generate-index` to create the master index

## Key Features

The app includes:

- Beautiful, readable typography inspired by Substack
- Reading progress tracking with localStorage
- Filter reviews by contest year
- Continue reading section for in-progress reviews
- Static site generation for fast performance
- Mobile-responsive design

## Next Steps

Once you've populated the content:
